DOM et Html :


Norme définissant un document du web.
• Est spécifiée en terme de DOM.
• HTML donne un sens à des éléments.
• Le DOM HTML fourni:
• un sens aux éléments du DOM (type de nodes): p, div, a, ...
• des APis relatives à la manipulation des éléments du
DOM: nouveaux objets dits "host" (window, document, navigator, ... ),
nouvelles propriétés (className, ... )
• des restrictions sur les éléments, par ex: les enfants d'un
ul doivent être des li.

Parmi les principales interfaces définies par le DOM
HTML, deux sont très utilisées lorsque l'on programme en JavaScript:
• Document, qui représente le contenu de la page web courante.
• Window, qui représente la fenêtre dans laquelle le document est chargé.

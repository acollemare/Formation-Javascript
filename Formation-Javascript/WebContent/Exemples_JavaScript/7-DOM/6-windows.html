Window :


Représente la fenêtre courante.
• il peut y avoir plusieurs objets window, par exemple dans
le cas d'onglets: chaque onglet à son objet window et le
browser a en plus son propre objet window.
• La propriété document d'une window pointe sur le
document DOM chargé dans cette fenêtre.
Expose des propriétés et des méthodes pour manipuler le container du document courant.
Référence :
• l'objet document : le document chargé
• l'objet location (lien de stockage) du document charge
• l'objet history : historique de navigation.
• l'objet navigator : qui contient des informations sur le browser

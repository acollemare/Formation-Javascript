Math :



- ABS()


Cette méthode renvoie la valeur absolue d'un nombre, il renvoie donc le nombre s'il est positif, son opposé (positif) s'il est négatif 
 x = Math.abs(3.26); 
//donne x = 3.26
  x = Math.abs(-3.26); 
//donne x = 3.26

- ceil()


Renvoie le plus petit entier supérieur ou égal à la valeur donnée en paramètre 
    x = Math.ceil(6.01); 
//donne x = 7
    x = Math.ceil(3.99); 
//donne x = 4

- floor()


La méthode floor() retourne le plus grand entier inférieur ou égal à la valeur donnée en paramètre. 
    x = Math.floor(6.01); 
//donne x = 6
    x = Math.floor(3.99); 
//donne x = 3

- round()


Arrondit à l'entier le plus proche la valeur donnée en paramètre.
Si la partie décimale de la valeur entrée en paramètre vaut 0.5, la méthode Math.round() arrondi à l'entier supérieur. 
    x = Math.round(6.01); 
//donne x = 6
    x = Math.round(3.80); 
//donne x = 4
    x = Math.round(3.50); 
//donne x = 4

- max(Nombre1, Nombre2)


max() renvoie le plus grand des deux nombres donnés en paramètre 
       var x = Math.max(6,7.25); 
   //donne x = 7.25
    var x = Math.max(-8.21,-3.65);
    //donne x = -3.65

- min(Nombre1, Nombre2)


Retourne le plus petit des deux nombres donnés en paramètre 
   x = Math.min(6,7.25); 
//donne x = 6
    x = Math.min(-8.21,-3.65,-55); 
//donne x = -8.21

- random()


La méthode random() renvoie un nombre pseudo-aléatoire compris entre 0 et 1. La valeur est générée à partir des données de l'horloge de l'ordinateur. 
  x = Math.random();

- sqrt(Valeur)


Renvoie la racine carrée du nombre passé en paramètre 
   x = Math.sqrt(9); 
//donne x = 3

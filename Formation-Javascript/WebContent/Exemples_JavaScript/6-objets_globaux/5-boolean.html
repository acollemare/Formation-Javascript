Boolean :


L'objet Boolean est un objet du noyau Javascript permettant de créer et de manipuler des valeurs de type booléennes.
Voici la syntaxe à utiliser pour créer une variable booléenne : Le paramètre peut-être soit une valeur (True ou False) 
    On peut créer un objet Boolean avec la syntaxe suivante :
    var b = new Boolean(valeur)

Asynchronous JavaScript And XML (AJAX) :



Utilise des objets JavaScript permettant de faire des requêtes asynchrones.
• L'idée et d'améliorer l'expérience utilisateur:
• en émettant des requêtes HTTP qui ne récupèrent que des fragments de données, ce qui évite des rechargements complets de pages.
• en émettant ces requêtes de manière asynchrone ce qui permet à l'utilisateur de continuer d'interagir avec le document chargé dans le navigateur, pendant que le navigateur fait la requête "en arrière plan".

Opérateurs :



Une expression est une unité valide de code qui se résoud comme une valeur.
Deux types d'expression :
- celles qui assignent une valeur à une variable, grâce à l'opérateur = 
 var x = 7
- celles qui se résolvent simplement comme une valeur, grâce à des opérateurs comme +,-,*, ... 
 var y = x + 5

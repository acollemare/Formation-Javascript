Opérateurs logiques :




&& :

ET (AND) logique. Retourne expr1 si elle peut être convertie à false sinon retourne expr2.
Donc avec des opérandes booléennes, && retourne true si les deux opérandes sont true, sinon retourne false.

     expr1 && expr2

|| :

OU (OR) logique. Retourne expr1 si elle peut être convertie à true, sinon retourne expr2.
Donc avec des opérandes booléennes Il retourne true si une des deux opérandes est true, sinon retourne false.

     expr1 || expr2

! :

NON (Non logique. Retourne false si son opérande peut être convertie à true, sinon retourne true.

     !expr1

Opérateurs de chaînes de caractères :




+ concatène



Dans la plupart des langages tels que PHP, la concaténation se fait avec le caractère ".".
En javascript, il suffit d'utiliser le caractère plus (+) ou d'utiliser la méthode concat(). deux valeurs String ensemble. "my" +"string" retourne "mystring" 

var chaine1 = "Bonjour ";
var chaine2 = "tout le monde";
var chaine3 = chaine1+chaine2;

ou 

var chaine1 = "Bonjour ";
var chaine2 = "tout le monde";
var chaine3 = chaine1.concat(chaine2);

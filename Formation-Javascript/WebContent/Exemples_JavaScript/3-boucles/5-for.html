FOR



Les boucle for, while et do autorisent l'usage de break pour sortir de l'itération.

Chacune de ces boucles possède une condition d'arrêt,
mais parfois il peut être souhaitable de sortir de la boucle pour d'autres raisons
(en cas d'erreur, si on a trouvé ce qu'on cherchait avant la fin d'une recherche, si l'utilisateur a décidé d'annuler une longue opération…)

.

 
for (i = 0; i < 10; i++)
{
   document.write(i + ' » ' );
}

for (i = 10; i > 0; i--)
{
   document.write(i + ' « ' );
}
document.write("0");

Associer dynamiquement un évènement à un élément :



Consiste à ajouter un évènement à une balise dans la page HTML alors que la page est déjà chargé ou à une balise HTML creé dynamiquement

Méthode 1 :

la plus ancienne de ces méthode consiste à appeler l'élément cible et de lui associer l'évènement par le nom de l'évènement et lui adjoindre le nom de la fonction.

document.getElementById('mon_element').onclick=nom_de_la_fonction


Compatible même avec les anciens navigateurs et simple a mettre en oeuvre
le problème de cette méthode est que ne l'on peut adjoindre qu'un seule évènement du meme type par élément

Methode 2 :

la deuxièmement méthode s'appel 'addEventListener' sa syntaxe est légèrement différente

document.getElementById('mon_element').addEventListener("click', nom_de_la_fonction, false);

Dans IE, il est nécessaire d'utiliser "attachEvent" plutôt que la méthode standard "addEventListener".
Pour que le code fonctionne dans IE<9,
if (var.addEventListener){
  element.addEventListener('click', modifieTexte, false);
} else if (element.attachEvent) {
  element.attachEvent('onclick', modifieTexte);
}


pour retirer l'evenement on utilise 'removeEventListener' 
document.getElementById('mon_element').removeEventListener("click', nom_de_la_fonction, false);




Methode 3 :

un événement dans une balise est consideré comme un attribut et pour cette troisieme methode nous allons utiliser 'setAttribute'

document.getElementById('mon_element').setAttribute('onclick','nom_de lafonction('paramètre',event)');


la méthode va insérer dans la balise l'évènement de la même manière que si on le faisait soit meme dans la balise on peut rajouter des paramètres et plusieurs fonction pour le même évènement sur l'élément.

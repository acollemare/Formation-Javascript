ERROR :



Créé des instances représentant une erreur
• Ces instances fournissent un ensemble standard d'informations utiles quand une erreur survient:
• message d'erreur.
• nom de la classe de l'erreur.
• stack trace.

 

    var error = new error ("The error message");
    console.log (error);
    console.log (error.message);
    console.log (error.stack);

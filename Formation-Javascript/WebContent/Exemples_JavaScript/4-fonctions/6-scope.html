SCOPE :



Une fonction peut accéder à toutes les variables et
fonctions définies dans le scope dans lequel elle est
définie.
• une fonction définie dans le scope global peut
accéder aux variables définies dans le scope global.
• une fonction (interne) définie dans une autre
fonction peut accéder aux variables définies dans la
fonction parent (externe) mais également à toutes les
variables auxquelles la fonction parent a accès.;

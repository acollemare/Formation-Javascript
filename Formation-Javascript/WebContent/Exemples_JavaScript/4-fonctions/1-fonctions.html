Fonctions :



Une fonction est un ensemble d'instructions qui effectuent une
tâche ou calculent une valeur.
- Les fonctions permettent de ré-utiliser facilement ces blocs
d'instructions et de les rendre paramétrables.
- Une fonction doit être définie quelque part dans le scope
depuis lequel elle doit être utilisée.

Description
Le mot clé return est utilisé dans le corps d'une function pour renvoyer le résultat d'un traitement.

 
function square(number) {
return number * number;
}

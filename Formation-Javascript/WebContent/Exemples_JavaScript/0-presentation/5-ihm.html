console.log



La fonction la plus courante est sans conteste console.log() qui permettra d'afficher un message,
une chaîne de texte, voire le contenu d'une variable (objet, tableau) de manière détaillée, via la console du navigateur(F12).
Les anciens navigateurs tels qu'Internet Explorer 6 et 7 n'en comportent pas. 

console.log("Kiwi"); 
console.log(3+3);

document.write



Ecrit le paramètre texte dans le document, à la position de l'appel du script.
Le paramètre texte peut être une chaîne de caractères classiques, du code HTML et même du javascript.
Pour afficher un texte sur plusieurs lignes, il faut concaténer la chaîne "<br />" au paramètre texte.
L'utilisation de writeln() permet pas de créer un saut de ligne HTML 
document.write("<TABLE border=1><TR>");

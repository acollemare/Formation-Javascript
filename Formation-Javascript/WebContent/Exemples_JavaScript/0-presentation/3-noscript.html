NOSCRIPT


Si un navigateur n' est pas capable d' exécuter le code JavaScript, prévoir un message entre les balises <NOSCRIPT>. 

<HTML>
<HEAD>
<TITLE>Test JavaScript</TITLE>
<SCRIPT language="JavaScript">
<!- Commentaire pour les navigateurs primitifs…
// Ici le code JavaScript
// Fin du code, fin de commentaire HTML: ->
</SCRIPT>
</HEAD>
<BODY>
Texte de la page
</BODY>
</HTML>

<NOSCRIPT>
Attention, cette page utilise normalement le
langage JavaScript qui n'existe pas ou est
désactivé sur votre navigateur.<BR />
</NOSCRIPT>

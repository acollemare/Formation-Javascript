Prototype :


c’est une liste de propriétés, attaché à un constructeur, qui va servir de « fallback » lorsque l’on cherche à accéder à une propriété manquante d’une instance dudit constructeur.
Lorsque l’on cherche à accéder à une propriété sur une instance, le langage  va d’abord regarder si l’instance contient cette propriété, et dans le cas contraire, va ensuite vérifier si le prototype de son constructeur la contient

var myClasse = function(){
     this.name = 'Romain';
   }
//Puis ensuite nous allons instancier cet objet :

   var o = new myClasse();
   console.log(o.name);
   
   
    myClasse.prototype.age = 25;
    console.log(o.age);

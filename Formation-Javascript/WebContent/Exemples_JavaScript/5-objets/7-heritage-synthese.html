Heritage Synthèse:


Lorsqu'un programme veut accéder à une propriété d'un objet :

1 JavaScript cherche dans l'objet si la propriété existe.
Si elle existe alors la valeur de la propriété est retournée.


2 Sinon si la propriété n'existe pas,JavaScript cherche la propriété dans l'objet référencé par
la propriété interne [[Prototype]]

 

3 et ainsi de suite.


La chaÎne de prototype permet donc de simuler un mécanisme ou

un objet [hérite des] / [partage les] propriétés de son prototype.


Une propriété trouvée dans la property chain est appelée une propriété héritée.

• Si la propriété n'est pas trouvée dans la prototype chain, la valeur undefined est retournée.

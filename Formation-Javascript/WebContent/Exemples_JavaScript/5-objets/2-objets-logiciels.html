Objet :


• stocke son state (état) dans des fields (champs, attributs de l'objet).
• expose son behaviour (comportement) à travers des methods (méthodes).


• En JavaScript:
• les fields sont des propriétés donc la valeur est type
primitif ou référence.

• les methodes sont des propriétés dont la valeur est une fonction.

 

 

En JavaScript un objet est un ensemble non ordonné de propriétés.
• Une propriété est une association entre un nom et une valeur.

• les propriétés sont comme des variables mais attachées à un objet.
• une méthode est une propriété dont la valeur est une fonction.


• JavaScript fournit des objets prédéfinis, le contexte d'exécution (par ex: le navigateur) définit des objets,
et il est possible de définir ses propres objets.

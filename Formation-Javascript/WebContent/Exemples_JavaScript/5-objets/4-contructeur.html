Deux étapes:
• définir la fonction constructeur, qui représente le type d'objet à créer.
Par convention, la première lettre de la fonction constructeur est en majuscule.
• créer une instance de l'objet en appelant la fonction constructeur précédée de new. 
    
   function Car(make, model, year) {
   this.make = make;
   this.model = model ;
   this.year = year;
}
